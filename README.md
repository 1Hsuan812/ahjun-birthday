# 🎂 阿君生日遊 | Ghibli Style Travel Guide

專為「阿君生日遊」打造的客製化單頁網站行程表！

## 🌟 功能亮點
- 🖼️ **原圖手繪海報整合**：點擊海報熱點或下方景點，即刻跳出景點卡片與 Google Maps 導航。
- 🌸 **淡雅櫻花飄落**：背景自帶輕柔視覺動畫，閱讀舒適不干擾。
- 🍏 **Apple 質感彈窗**：點擊景點帶有淡入與縮放效果。
- 📍 **行程完成打卡 (LocalStorage)**：離線/重新整理網頁後依然自動保存打卡進度。
- 📱 **手機響應式**：支援雙指放大檢視海報，完美適配 iPhone 與 Android 手機。
- 📤 **一鍵原生分享**：方便直接分享至 LINE、IG 或 AirDrop。

## 📁 專案結構

```
阿君生日遊/

```

## 🚀 GitHub Pages 部署步驟
1. 將專案內容上傳至您的 GitHub Repository。
2. 進入 Repository -> **Settings** -> **Pages**。
3. Source 選擇 `main` 分支並儲存。
4. 幾秒鐘後即可取得專屬網址分享給朋友！
